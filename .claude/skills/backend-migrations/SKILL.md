## When to use this skill:

- When creating new database migration files
- When modifying database schema (adding/removing tables, columns, indexes)
- When working on files in `/migrations/`, `/db/migrate/`, `/prisma/migrations/`, or similar directories
- When writing rollback/down methods for migrations
- When adding database indexes or constraints
- When planning schema changes for zero-downtime deployments
- When renaming tables or columns
- When managing foreign key relationships in migrations
