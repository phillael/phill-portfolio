## When to use this skill:

- When creating or modifying database model files
- When working on files in `/models/`, `/entities/`, `/schemas/`, or `/prisma/schema.prisma`
- When defining model relationships (hasMany, belongsTo, manyToMany)
- When adding database constraints (NOT NULL, UNIQUE, foreign keys)
- When choosing appropriate data types for columns
- When implementing model-level validations
- When adding timestamps (createdAt, updatedAt) to models
- When setting up cascade behaviors for relationships
- When working with Prisma, Sequelize, TypeORM, or other ORM files
