## When to use this skill:

- When creating or modifying UI components (.tsx, .jsx, .vue, .svelte files)
- When adding interactive elements (buttons, links, forms, modals)
- When implementing keyboard navigation (onKeyDown handlers, tabIndex)
- When adding ARIA attributes (aria-label, aria-expanded, aria-hidden, role)
- When writing alt text for images
- When creating form inputs with proper labels
- When implementing focus management in modals or dynamic content
- When checking or improving color contrast ratios
- When adding focus-visible styles for keyboard users
- When structuring heading hierarchy (h1-h6)
- When building navigation menus or skip links
