## When to use this skill:

- When writing CSS classes or Tailwind utility classes
- When working on .css, .scss, .sass, .less, or .module.css files
- When adding inline styles or styled-components
- When implementing design tokens (colors, spacing, typography)
- When using Tailwind CSS utilities in component files
- When creating or modifying globals.css or theme files
- When implementing dark mode or theme switching
- When optimizing CSS for production (purging unused styles)
- When maintaining consistent spacing and color systems
- When avoiding framework style overrides
