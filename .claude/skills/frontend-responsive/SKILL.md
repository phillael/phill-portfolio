## When to use this skill:

- When implementing responsive layouts with CSS or Tailwind
- When using breakpoint prefixes (sm:, md:, lg:, xl:, 2xl:)
- When adding media queries to CSS files
- When converting desktop layouts to mobile-friendly designs
- When implementing fluid/percentage-based widths
- When ensuring touch targets are at least 44x44px
- When optimizing images for different screen sizes
- When adjusting typography for mobile readability
- When testing layouts across mobile, tablet, and desktop
- When implementing responsive navigation (hamburger menus, mobile nav)
- When prioritizing content for smaller screens
