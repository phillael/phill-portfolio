## When to use this skill:

- When validating form inputs on the frontend
- When validating API request bodies or query parameters
- When implementing Zod, Yup, or other schema validation
- When sanitizing user input to prevent injection attacks
- When providing field-specific validation error messages
- When implementing type and format validation
- When validating business rules (dates, ranges, permissions)
- When setting up validation middleware in APIs
- When implementing allowlist vs blocklist patterns
- When ensuring validation consistency across entry points
- When preventing XSS, SQL injection, or command injection
