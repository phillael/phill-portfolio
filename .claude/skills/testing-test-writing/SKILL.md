Apply when using Jest, React Testing Library, Vitest, or other testing frameworks, when mocking dependencies, or when deciding what to test.
---

## When to use this skill:

- When creating test files (.test.ts, .test.tsx, .spec.js, etc.)
- When writing unit tests for functions or components
- When writing integration tests for API endpoints
- When writing end-to-end tests with Playwright or Cypress
- When using Jest, React Testing Library, Vitest, or similar
- When mocking external dependencies or API calls
- When deciding which tests to write for a feature
- When naming test cases descriptively
- When focusing tests on behavior rather than implementation
- When keeping test execution fast and reliable
- When testing critical user flows and core functionality
