# Audio Files

Place your audio files (MP3 format) in this directory.

The music player expects files to be named according to the paths specified in `/src/data/tracks.json`.

## Expected Files

Based on the current track configuration:

- `cosmic-dance.mp3`
- `neon-groove.mp3`
- `electric-dreams.mp3`
- `forest-awakening.mp3`
- `ocean-depths.mp3`
- `mountain-summit.mp3`
- `code-coffee.mp3`
- `midnight-debugging.mp3`
- `async-dreams.mp3`
- `final-deploy.mp3`

## File Specifications

- Format: MP3
- Recommended bitrate: 128-192 kbps
- Total size should be approximately 40MB for all tracks
- Files will be streamed with metadata preloading for performance
